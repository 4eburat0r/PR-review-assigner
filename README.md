# Avito Backend Trainee - Сервис назначения ревьюеров для Pull Request'ов

Данный репозиторий представляет собой каркас сервиса на Go + PostgreSQL, реализующий тестовое задание.

## Реализованы 2 дополнителных задания:

1. Добавлен простой эндпоинт статистики
2. Добавлен метод массовой деактивации пользователей

## Запуск проекта

1. git clone https://github.com/4eburat0r/pr-review-assigner.git
2. cd pr-review-assigner
3. make run
2. Сервер будет доступен по адресу: <http://localhost:8080>
