# Avito Backend Trainee - Шаблон выполненного проекта

Данный репозиторий представляет собой сгенерированный рабочий каркас проекта на Go + PostgreSQL, реализующий тестовое задание.

## Что включено

- API, реализованный в соответствии с загруженной OpenAPI спецификацией
- Docker + docker-compose для простого запуска
- Автоматическое применение миграций БД
- Базовые модульные тесты
- Реализованы 2 дополнителных задания:

1. Добавлен простой эндпоинт статистики
2. Добавлен метод массовой деактивации пользователей

## Запуск проекта

1. Выполните команду: `docker-compose up --build`
2. Сервер будет доступен по адресу: <http://localhost:8080>
